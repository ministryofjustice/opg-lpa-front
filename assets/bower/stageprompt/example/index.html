Stageprompt example

For debugging and testing, the Google Analytics Debugger for chrome is very useful.

Setup

This page has been set up using the default config:

                    
<script type="text/javascript">
    $(function(){
        GOVUK.performance.stageprompt.setupForGoogleAnalytics();
    });
</script>

More on configuration types can be found in the documentation for Stageprompt.
